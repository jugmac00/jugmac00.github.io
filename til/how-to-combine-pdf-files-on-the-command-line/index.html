How to Combine PDF Files on the Command Line

April 24, 2022

Without further ado…

pdfunite in-file-1.pdf in-file-2.pdf outfile.pdf

Simple as that.

How to get pdfunite?#

I had it already installed as a dependency when I installed calibre, but you can also install it via…

sudo apt install poppler-utils

poppler-utils is a collection of PDF utils.
