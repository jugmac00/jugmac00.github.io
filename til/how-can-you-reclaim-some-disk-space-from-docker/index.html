How Can You Reclaim Some Disk Space From Docker

May 5, 2021

If you use Docker containers, and who doesn’t these times, you need to take care that Docker does not accumulate a lot of cruft.

remains of docker run#

If you start a container via docker run, and without the highly recommended option --rm, the exited Docker container will still leave some remains on your system.

e.g. docker image rm redis
